脸疼 App（脸厚不疼）

脸疼（FaceTeng.App）和 不占座 是南京石匠网络科技有限公司（上海分公司）的产品，老板马骏是新疆人，毕业于南京金陵学院，擅长威胁员工、拖欠工资、违法辞退女员工、低缴社保、偷税漏税、趁着疫情模(chao)仿(xi)竞品（武汉 Timing）。

南京石匠已被法院和劳动仲裁判决赔偿，被社保局查处（按最低基数交几百块社保），技术总监、运营负责人、运营部门集体离职了，技术也裁了，不占座 没火起来，又做了 脸疼 从学生身上收钱，不知道这公司脸疼不疼。

司法判决

南京中级法院：（2019）苏01民特282号

某女生前端工程师工作优秀，石匠公司却拖欠工资，并指使后端工程师做出考核不合格的评判，辞退不给赔偿。劳动仲裁（宁鼓劳人仲案字[2019]第756号）判决公司需支付拖欠的工资、违法辞退赔偿2万多元。公司拒不赔偿，起诉到法院，判公司败诉，已在网上公示：民事裁定书 - 南京石匠网络科技有限公司。

宁鼓劳人仲案字[2019]第755号

某女生运营工作优秀，经常获得月度绩效奖金，石匠公司却以不合格为由让人主动辞职，否则开除不给赔偿，并拖欠工资。劳动仲裁判决公司需支付拖欠的工资、违法裁员赔偿2个月工资。

宁鼓劳人仲案字[2019]第754号

技术总监垫付的 GitHub 费用被拖欠，离职索要加班费被老板威胁：是我们让你加班的吗？如果要加班费的话，公司就到猎头圈造谣你是不称职被开除的。劳动仲裁判决公司需支付拖欠的 GitHub 费用。

威胁员工

南京石匠网络科技有限公司的老板马骏是新疆人，天使轮创业加班严重，技术总监离职要加班费，被老板威胁：是我们让你加班的吗？你有证据吗？如果要加班费的话，公司就到猎头圈造谣你是不称职被开除的。录音下载：https://share.weiyun.com/57Pt53X

结果：加班费果然没证据，因为公司指使在职党员等人签字证明：公司有加班申请制度（实际之前没有申请制度），还伪造了2018年的加班单，欺骗司法部门。 公司还真去造谣了，说技术总监离职索要期权变现。
